## Public Data from StatsNZ

This repository contains public (mostly spatial) datasets from [StatsNZ](https://stats.govt.nz). Unless specified otherwise the data is licensed under [Creative Commons Attribution 4.0 license](https://creativecommons.org/licenses/by/4.0/) and obtained by export from the [StatsNZ DataFinder](https://datafinder.stats.govt.nz/) website. The data have not been modified other than a) unpacking from the ZIP files into the corresponding directories named by the dataset and b) compressing individual large files.
